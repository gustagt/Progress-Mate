# Progress Mate
> O projeto permitirá que o usuário registre seu progresso em diferentes categorias (manhwa, série, manga, anime), visualize o que está acompanhando e atualize o progresso de forma eficiente. O sistema será baseado em uma arquitetura web responsiva, com APIs para gerenciar dados e autenticação de usuários.
